  <h2>Part 1: Implementing input methods in Java</h2>
<p>This Java program is menu-driven and implements various input methods in Java. The program takes user inputs from the following input methods:</p>
<ul>
	<li>Command Line Args</li>
	<li>Scanner</li>
	<li>BufferedReader</li>
	<li>DataInputStream</li>
	<li>Console</li>
</ul>

<h2>Part 2: Simple menu-driven calculator</h2>
<p>This Java program is a simple menu-driven calculator that performs the following functions:</p>
<ul>
	<li>Addition</li>
	<li>Subtraction</li>
	<li>Multiplication</li>
	<li>Division</li>
	<li>Square Root</li>
	<li>Power</li>
	<li>Mean</li>
	<li>Variance</li>
</ul>

<p>The calculator functions are implemented in a separate Calculator class that includes all related functions inside that class.</p>

<h3>Usage</h3>
<p>The program can be run using the following command:</p>
<code>java Main</code>
<p>The program will then display a menu and prompt the user to select an operation.</p>

<h3>Input Methods</h3>
<p>The user can select the input method by entering the corresponding number from the menu. The program will then prompt the user to enter the required inputs using the selected input method.</p>

<h3>Mean Calculation</h3>
<p>The mean calculation function reads numbers from the keyboard, summing them in the process until the user enters the string “end”. It then stops input and displays the average of the numbers.</p>
